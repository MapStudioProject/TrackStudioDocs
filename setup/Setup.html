Setup

Setup

  • Make sure you have Net5.0 installed on your PC.
  • Make sure you have C++ resdist.
  • Make sure you have Direct X

Direct X is optional but will help make texture encoding much faster.

Now that is done you are ready to open the tool. Open MapStudio.exe for windows.

For other platforms run dotnet MapStudio.dll in the cmd.

With this you want to configure your game paths.

If you don't have a game dump you can still edit but won't have visuals for map objects)

Now you can either file - open or drag/drop files to edit.
